# Invoice Management System

A full-stack Invoice Management System built with React, Express, and SQLite.

## Features
- **Authentication**: JWT-based signup and login.
- **Invoices**: Create, View, Update, and Delete invoices.
- **Search & Filter**: Search by client/invoice number, filter by status.
- **Responsive UI**: Clean and premium dashboard designed with Vanilla CSS.

## Tech Stack
- **Frontend**: React (Vite), Axios, React Router, Lucide React.
- **Backend**: Express.js, better-sqlite3, JWT, bcryptjs.
- **Database**: SQLite.

## How to Run Locally

### Prerequisites
- Node.js (v18+)
- npm

### 1. Setup Backend
```bash
cd backend
npm install
npm run dev
```
*Creates a `database.sqlite` file and starts server on port 5000.*

### 2. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```
*Starts development server on port 5173.*
